HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding insdicates how much space we want around the content inside an element.
  • Colors can also be added with hexadecimal codes besides typing their actual names
  • Using class selector all the elements with same value/name will show style change

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • Variables are containers that allow us to hold data in our code
  • Controlflow is the order in which computer executes code in a script
  • Functions are set of instructions that tell computer how to perform a task
  • Functions need to be defined and called out so they can work
